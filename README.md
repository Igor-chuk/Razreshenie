## Быстрый старт
1. Скачайте последний релиз из [Releases](../../releases)
2. Распакуйте в папку без пробелов (например, `C:\Razreshenie\`)
3. Запустите `Razreshenie.exe` от имени администратора
4. Нажмите "Запустить"
   
    4.1. Если вы не видите кнопок снизу, разверните приложение на весь экран
